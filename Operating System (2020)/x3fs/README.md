# X³ File System (g3fs.v2)

Thanks a lot to oyiadin's [g3fs](https://github.com/oyiadin/Schoolworks/tree/master/Operating%20System%20(2019)/g3fs)

## What's new

- bug fixes
- better CLI
- fcb timestamp
- file seek
- file rename
- lsof support
- open file in different mode
- symlink support

## What's missing
